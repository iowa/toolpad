# @iowas/toolpad

MUI-powered dashboard components and utilities for Next.js and Drizzle ORM.

## Installation

```bash
pnpm add @iowas/toolpad
```

## Architecture

**@iowas/toolpad/core**
Foundational UI layer providing Material UI dashboard layouts, application providers, and common interface patterns.

**@iowas/toolpad/node**
A collection of infrastructure utilities for database connection pooling, URL query string management, and standardized data grid types.

## Demo Implementation

Reference implementation is available in `src/app`, showcasing layout composition and provider configuration for the Next.js App Router.
